# gulp-sass-babel

Example gulp setup with SCSS compilation and Babel transpilation.

Configured to watch for changes in `app/js` and `app/scss`

### Instructions for use
- clone repo
- run `npm install`
- run `./node_modules/gulp/bin/gulp.js watch`
